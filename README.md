# 清华社英语在线

## 项目概述

这是一个用于清华社英语在线网站（www.tsinghuaelt.com）的用户脚本，提供自动化答题功能。该脚本支持多种题型，包括填空题、选择题、判断题、拖拽题、视频题等，能够自动完成英语学习任务。

脚本目前存在一些问题，包括但不限于：

- 对一些特殊题型的支持不完善
- 在某些情况下可能会出现错误或异常

> 已发现的问题放在该文档末尾，请注意查看，如有疑问请[联系我](#contact-email)

## 安装和使用

### 安装步骤

1. 安装脚本猫浏览器扩展，再选择Tampermonkey，优先使用脚本猫哦
2. 创建新用户脚本
3. 将main.js内容复制到脚本编辑器中
4. 保存并启用脚本
5. 访问清华社英语在线网站（www.tsinghuaelt.com）

### 使用方法

1. 登录清华社英语在线网站
2. 进入需要完成的课程页面
3. 脚本会自动检测并开始答题
4. 观察控制台日志了解执行状态
5. 完成所有题目后，脚本会自动停止

## 功能特性

### 支持的题型

1. **填空题** (auto_tiankong)
   - 自动填写文本答案
   - 支持文本输入框和文本域

2. **录音题** (auto_lytk)
   - 自动识别语音输入
   - 支持语音填写填空题

3. **单选题** (auto_danxuan)
   - 自动选择正确答案
   - 支持角色扮演类题目

4. **选择题** (auto_dropchoose)
   - 自动选择正确答案
   - 支持多选题

5. **拖拽题** (auto_drag)
   - 智能拖拽操作
   - 自动滚动和位置调整

6. **视频题** (auto_video)
   - 视频播放器控制
   - 自动播放和提交

7. **多选题** (auto_duoxuan)
   - 多选题目处理

8. **判断题** (auto_judge)
   - 正确/错误判断（不支持三个选项判断题）
   - 智能答案匹配

### 核心功能

- **自动化答题**: 自动识别并完成各种题型
- **智能等待**: 可配置的等待时间，适应不同网络环境
- **详细日志**: 丰富的控制台日志输出（按F12打开控制台查看）

## 配置说明

### 时间配置

```javascript
const submitDelay = 3000;       // Submit之后的等待时间
const pageNextDelay = 5000;     // 换页之后的等待时间
const inputDelay = 500;         // 输入之后的等待时间
```

### 用户配置

```javascript
let user_config = {
    'autodo': [                 // 启用的自动化功能
        'auto_tiankong',        // 填空题
        'auto_luyin',           // 录音题
        'auto_danxuan',         // 单选题
        'auto_drag',            // 拖拽题
        'auto_video',           // 视频题
        'auto_duoxuan',         // 多选题
        'auto_judge'            // 判断题
    ],
    'autotryerr': true,        // 自动重试错误
    'autostop': false,          // 自动停止
    'autorecord': true,         // 自动录音
    'delay': 10000              // 用户设置的等待时长
};
```

## 问题及注意事项

### 已发现问题

1. **对一些特殊题型的支持不完善**
   - 判断题只能判断正确或错误，不能出现第三个判断项

2. **在某些情况下可能会出现错误或异常**
   - 网络延迟导致的答题失败、题目结构变化导致的识别错误等

3. **其他问题等**
   - 所有已发现的问题我将放在https://github.com/FmCoral/tsinghuaelt/issues.md

### 注意事项

1. **答题过程中人工操作可能会影响脚本运行**
   - 进行拖拽题答题时，弹窗不要覆盖答题框和选项框，并且答题时不要移动鼠标，否则会引起拖拽失败
   - 答题时最好不要切换或者最小化页面，防止脚本运行不稳定

2. **脚本未包括题型说明**
   - 有评分录音，包括系统打分的录音和角色扮演的录音题型不能进行自动答题
